📱 COMO ACESSAR

1️⃣ REDE LOCAL (Escritório Viza)

Pré-requisito: Estar conectado na mesma WiFi (Rede_Viza)

Acesso:
http://iluminacao-viza.local

Com HTTPS (após configurar):
https://iluminacao-viza.local

SSH:
ssh pi@iluminacao-viza.local

─────────────────────────────────────────────

2️⃣ VPN WIREGUARD (Acesso Remoto)

De qualquer lugar (celular com 4G, casa, etc)

Acesso:
http://10.0.0.1
https://10.0.0.1 (com HTTPS)

Configurar:
sudo ./wireguard_setup.sh

─────────────────────────────────────────────

3️⃣ VIA IP DIRETO (alternativa)

Se .local não funcionar:

Descobrir IP:
hostname -I
Resultado exemplo: 192.168.1.150

Acesso:
http://192.168.1.150

🔧 ALTERAR HOSTNAME DEPOIS (se necessário)

Se já instalou e quer mudar depois:

Opção A - Script Automático:
sudo ./scripts/setup_hostname.sh

Opção B - Manual:
sudo hostnamectl set-hostname iluminacao-viza
sudo nano /etc/hosts
# Alterar linha: 127.0.1.1 iluminacao-viza
sudo reboot

🐛 TROUBLESHOOTING

PROBLEMA: iluminacao-viza.local não funciona
SOLUÇÃO: Usar IP direto (hostname -I)

PROBLEMA: Hostname não mudou
SOLUÇÃO: sudo reboot (reiniciar Raspberry)

PROBLEMA: mDNS não suporta .local (Windows antigo)
SOLUÇÃO: Instalar Bonjour Print Services
https://support.apple.com/kb/DL999

PROBLEMA: Quer voltar para raspberrypi.local
SOLUÇÃO: sudo ./scripts/setup_hostname.sh
         Informar: raspberrypi
